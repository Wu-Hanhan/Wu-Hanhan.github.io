**Fall 2025** — I served as a Graduate Teaching Assistant for three courses in the ECE department at George Mason University.

### ENGR 107: Introduction to Engineering
- Assisted in course management and graded assignments and assessments for a class of 40 students.

### ECE 240: C Programming for Engineers
- Graded programming assignments and exams for 98 students.
- Provided technical support on C programming concepts, debugging strategies, and coding best practices.

### ECE 554: Machine Learning for Embedded Systems
- Graded coursework and project deliverables for 38 graduate students.
- Attended project proposal and progress presentations, offering technical feedback on model design, embedded deployment strategies, and experimental methodology.
